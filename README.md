# Python Trade Copier (MT5)

A Python-based MetaTrader 5 trade copier for educational purposes.

This project copies open trades from a MASTER account to a SLAVE account using the official MetaTrader5 Python API.

---

## Features
- Copy open trades from MASTER to SLAVE account
- Secure configuration handling
- Uses official MT5 Python API
- Clean and simple project structure
- GitHub-safe (no credentials in code)

---

## Requirements
- Python 3.8 or higher
- MetaTrader 5 terminal installed
- MetaTrader5 Python package

---
