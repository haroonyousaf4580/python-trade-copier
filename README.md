# Python Trade Copier (MT5)

A Python-based MetaTrader 5 trade copier for educational purposes.

## Features
- Copy trades from MASTER to SLAVE account
- Secure config handling
- MT5 API based
- Clean and simple structure

## Requirements
- Python 3.8+
- MetaTrader 5 installed
- MetaTrader5 Python package

## Installation
```bash
pip install MetaTrader5
## Usage
1. Copy `config.example.py` to `config.py`
2. Fill in your MT5 account credentials
3. Run the script:

```bash
python main.py
